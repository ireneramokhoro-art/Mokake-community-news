# Mokake-community-news
This website was created to share local news, events and important information within the community. Its main purpose is to keep people informed and connected. It includes five pages: Home, News, Events, Videos and Contact. Each page provides different information such as community stories, upcoming activities and video interviews. This website was developed using HTML and CSS, with images and a navigation menu to make it easy to use. Thisprojecthelped me improve my web desidn skills and understand how websites can be used to communicate information effectively.
